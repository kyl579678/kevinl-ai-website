Defect iDoctor

A semiconductor defect diagnosis platform demo built to showcase AI-driven defect analysis capabilities for advanced wafer manufacturing.

Features

  • Interactive wafer defect map visualization (300mm wafer, scatter/cluster/ring/scratch/edge patterns)
  • FDC/SPC production history browser with out-of-spec detection
  • Trend analysis with daily/hourly defect volume charts
  • Per-tool defect attribution
  • Dashboard with key manufacturing KPIs

Tech Stack

  • Backend: FastAPI (Python) serving REST APIs
  • Frontend: Vanilla JS + Chart.js + SVG
  • Data: Synthetic datasets (100 wafers, 27k+ defects, 2k+ production records)

Data Pipeline

Synthetic data generators create realistic defect distributions and FDC parameter traces with configurable patterns. ~5% of production parameters are intentionally out-of-spec to simulate excursion scenarios.
